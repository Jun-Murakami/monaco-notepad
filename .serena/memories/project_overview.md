Monaco Notepad is a Wails v2 desktop note app for macOS/Windows with Monaco Editor and Google Drive sync. Backend is Go in a mostly single `backend` package. Frontend is React 19 + TypeScript + Vite + MUI with business logic in custom hooks under `frontend/src/hooks` and UI in `frontend/src/components`. `App.tsx` is the main composition root and passes state/handlers down via props; project guidance explicitly prefers props drilling over external state libraries.
