Frontend conventions: keep business logic in custom hooks, keep components mostly UI-only, and avoid external state libraries like Redux/Zustand unless architecture is intentionally changed. TypeScript files use tabs. Types live in `frontend/src/types.ts` plus generated Wails types. Backend conventions: use `package backend`, prefer MessageCode-based user-facing messages, inject services via `App`, use `New*Service()` factories, and keep comments in Japanese.
