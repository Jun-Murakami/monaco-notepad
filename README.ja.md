# Monaco Notepad

[English](README.md) | **日本語**

**OSのメモ帳×VSCode×Evernoteを足して10で割ったようなアプリ**

Monaco Editor（VS Codeと同じエンジン）を搭載したプログラマー向けノートパッドです。ファイルを開いて直接編集したり、クラウドノートに変換して複数デバイスで同期したりできます。

## 特徴

### 💡 ハイブリッドな使い方

- **ローカルファイルの直接編集** — ファイルを開いてそのまま編集できます
- **クラウドノート変換** — ローカルファイルをクラウドノートに変換して、Evernoteのように複数デバイスで同期
- **プライベートストレージ** — クラウドノートはあなたのGoogleドライブを使用（アプリ専用フォルダにのみアクセス）
- **オフライン対応** — ネットワークなしでも完全に動作

### 📝 エディタ

- **Monaco Editor** — 50種類以上の言語でシンタックスハイライト
- **自動保存** — 3秒間の入力停止後に自動保存
- **カスタマイズ可能** — フォントファミリー、フォントサイズ、エディタテーマを設定可能
- **便利な機能** — ワードラップ、ミニマップの切り替え
- **ダーク/ライトモード** — スムーズなテーマ切り替え

### ⚡ 分割エディタ

- **サイドバイサイド表示** — 2つのノートを同時に編集
- **右クリックメニュー** — **左ペインで開く** / **右ペインで開く**
- **自動分割モード** — コンテキストメニューから自動的に分割モードを起動
- **重複検知** — 同じファイルが両方のペインに表示されそうな場合、他のペインは次の利用可能なアイテムを読み込み
- **状態保持** — アプリ再起動時も分割状態を維持
- **カラーコード** — ペインインジケーターで色分け

### 📖 Markdownプレビュー

- **ライブプレビュー** — GitHub Flavored Markdown (GFM) をサポート
- **シンタックスハイライト** — コードブロックに対応
- **配置設定** — 左側または右側に配置可能（設定で変更）

### 📁 ノート管理

- **基本操作** — ノートの作成、編集、アーカイブ、削除
- **フォルダ整理** — ドラッグ&ドロップで並び替え
- **アーカイブシステム** — 復元機能付き
- **全文検索** — すべてのノートとファイル内容を検索、マッチ箇所をナビゲート

### 💾 ローカルファイル編集

- **ファイルを開く** — ローカルファイルを直接開いて編集
- **保存/名前を付けて保存** — 編集内容を保存
- **未保存の変更通知** — 保存されていない変更があることを表示
- **クラウドノート変換** — ローカルファイルをクラウドノートに変換
- **ドラッグ&ドロップ** — ファイルをドラッグ&ドロップで開く

### ☁️ Google Driveの同期

- **OAuth2認証** — 安全なGoogle認証
- **自動バックグラウンド同期** — Changes APIを使用した自動同期
- **コンテンツハッシュベース** — 変更検知による効率的な同期
- **非同期操作キュー** — アップロードとダウンロードの非同期処理
- **ビジュアルステータス** — ステータスバーに同期状態を表示

### 📊 ステータスバー

- **文字数、行数、カーソル位置** を表示
- **通知エリア** — 履歴（最大1,000件）
- **クイックアクセスボタン** — 分割エディタ、Markdownプレビュー、Google Drive、設定
- **バージョンアップ通知**

## キーボードショートカット

| ショートカット | 動作 |
|---|---|
| `Ctrl/Cmd + N` | 新しいノート |
| `Ctrl/Cmd + O` | ファイルを開く |
| `Ctrl/Cmd + S` | ファイルを保存 |
| `Ctrl/Cmd + Alt + S` | 名前を付けて保存 |
| `Ctrl/Cmd + W` | ファイルを閉じる / ノートをアーカイブ |
| `Ctrl/Cmd + Tab` | 次のノート |
| `Ctrl/Cmd + Shift + Tab` | 前のノート |

## はじめ方

1. お使いのプラットフォーム（macOS / Windows）の最新リリースをダウンロード
2. アプリを起動 — 追加のセットアップは不要です
3. オプション：クラウド同期のためにGoogleアカウントを接続

## 技術スタック

| レイヤー | 技術 |
|---|---|
| バックエンド | Go + [Wails v2](https://wails.io/) |
| フロントエンド | React 19 + TypeScript + Vite |
| エディタ | [Monaco Editor](https://microsoft.github.io/monaco-editor/) |
| UI | Material UI (MUI) v7 |
| 同期 | Google Drive API v3 |
